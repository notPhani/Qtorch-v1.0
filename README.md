# Qtorch-v1.0
QtorchX: High-fidelity quantum simulation using the QNaF (Quantum Noise as Fields) formalism. Models noise as a 7D manifold with bidirectional field-circuit interaction. Features differentiable, trainable Decoherence Projection Matrices (BPO) and Baseline Pauli Offsets (BPO) for calibrating simulations to real hardware.
